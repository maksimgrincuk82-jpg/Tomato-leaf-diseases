# Tomato-leaf-diseases
ІНТЕЛЕКТУАЛЬНА СИСТЕМА НЕЙРОМЕРЕЖЕВОГО ДІАГНОСТУВАННЯ ХВОРОБ ЛИСТЯ ТОМАТА ЗА ФОТОЗОБРАЖЕННЯМИ

Наукова робота присвячена розробці та дослідженню інформаційної технології для автоматизованої класифікації захворювань листя томату за зображеннями. Актуальність дослідження обумовлена зростаючою потребою у впровадженні систем штучного інтелекту в аграрний сектор, зокрема для моніторингу стану рослин і своєчасного виявлення патологій, що дозволяє зменшити втрати врожаю та підвищити ефективність агровиробництва.
Метою роботи є створення та експериментальна перевірка інформаційної технології, здатної автоматизовано розпізнавати основні види захворювань томатів за зображеннями листя з використанням глибоких нейронних мереж.
У межах дослідження проведено аналіз сучасних нейромережевих методів діагностики агрокультур, реалізовано та порівняно дві архітектури глибокого навчання: Vision Transformer та MobileNetV2. За результатами експериментів власноруч модифікована модель Vision Transformer продемонструвала найвищі показники якості класифікації а саме F1-score 97,32. 
Розроблена інформаційна технологія може бути використана як основа для побудови інтелектуальних систем моніторингу стану агрокультур у теплицях, фермерських господарствах і дослідницьких лабораторіях.
Ключові слова: комп’ютерний зір, Vision Transformer, MobileNetV2, класифікація зображень, захворювання томатів, штучний інтелект, агротехнології.
